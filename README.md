# SarkarSaathi

SarkarSaathi is a voice-first AI assistant that helps citizens understand and fill government forms without needing to read or type.

## Problem

Government forms are often complex and difficult to understand, especially for low-literacy and rural citizens. Many depend on middlemen or cyber cafes for assistance.

## Solution

SarkarSaathi enables users to:

- Upload government forms (PDF/Image)
- Receive voice guidance in regional languages
- Fill forms through speech interaction
- Validate inputs in real time
- Generate a structured filled PDF for submission

## Key Features

- OCR-based form parsing
- LLM-powered conversational guidance
- Multi-language speech-to-text and text-to-speech
- Session persistence
- Structured PDF generation
- Accessibility-focused design
- Secure data handling and automatic expiration

## Architecture

The system follows a modular architecture including:

- API Gateway
- Session Manager
- Form Processor
- Voice Engine
- PDF Generator

See:
- `requirements.md`
- `design.md`

## Impact

SarkarSaathi aims to improve accessibility to governance services by reducing literacy barriers and eliminating dependency on intermediaries.
